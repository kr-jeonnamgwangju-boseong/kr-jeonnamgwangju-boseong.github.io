# 보성군 · 전남광주통합특별시

> 보성군 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-jeonnamgwangju-boseong.github.io/**

보성군은 한반도 남해안, 옛 전라남도 중남부에 자리한 '녹차의 수도'로 불리는 고장이다. 군청 소재지는 보성읍이며 2읍 10면을 관할하고, 면적은 약 663㎢에 이른다. 활성산 일대의 산비탈을 계단식으로 수놓은 대한다원 녹차밭은 국내 최대 규모로, 삼나무 가로수길과 함께 수많은 영화·드라마·광고의 배경이 되며 한국을 대표하는 풍경으로 자리 잡았다. 남쪽 득량만의 청정 갯벌에서 나는 벌교 꼬막은 수산물 지리적표시 제1호로 등록될 만큼 명성이 높고, 소설 《태백산맥》의 무대인 벌교읍은 근대 문학기행의 명소다. 서편제 판소리의 본향이자 율포해수욕장·제암산 철쭉·오봉산 구들장 채석장까지, 차·바다·소리·문학이 어우러진 남도의 정취를 고루 품은 지역이다.
